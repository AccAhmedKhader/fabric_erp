# FabricERP Accounting System

## Phase 1: Identity & Authentication

### Enterprise ERP System for Egyptian Textile and Garment Trading Companies

## Technology Stack
- Python 3.13
- Django 5.x
- PostgreSQL
- Bootstrap 5
- Docker (Optional)

## Features (Phase 1)
### User Management
- Complete user CRUD operations
- Role-based access control
- Permission management
- User activation/deactivation

### Authentication
- Secure login/logout
- Password reset via email
- Email verification
- Remember me functionality
- Session timeout
- Account lockout after failed attempts
- Login audit trail

## Installation
### Prerequisites
- Python 3.13+
- PostgreSQL 15+

### Setup
1. Clone the repository
2. Create virtual environment
3. Install dependencies
4. Configure environment variables
5. Create database
6. Run migrations
7. Create superuser
8. Run development server
