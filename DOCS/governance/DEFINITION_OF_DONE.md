# Definition Of Done

لا تعتبر المهمة منتهية إلا إذا:

✓ الكود مكتمل.

✓ جميع الاختبارات ناجحة.

✓ Coverage ≥ 90%.

✓ Documentation محدثة.

✓ API موثقة.

✓ Migration مضافة.

✓ Logging موجود.

✓ Security Review.

✓ Architecture Review.

✓ Code Review Approved.

✓ لا توجد Critical Bugs.

✓ لا توجد High Vulnerabilities.

✓ تم تحديث Roadmap عند الحاجة.
